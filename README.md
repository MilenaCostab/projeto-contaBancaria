🔗 **Projeto Banco - Sistema de Gestão de Contas Bancárias**

Sistema de gerenciamento bancário desenvolvido com **ASP.NET Core (C#)** no back-end e **HTML/CSS/JS** puro no front-end.  
O projeto simula operações reais de um banco, incluindo diferentes tipos de contas, sistema de login para clientes e administradores, além de funcionalidades de rendimento.



🔗 **Funcionalidades**

🔹 **Para Clientes**
- Cadastro – Auto-registro para novos usuários.  
- Login – Acesso seguro com validação de CPF e senha.  
- Transações – Saques, depósitos e transferências entre contas.  
- Caixinha (Poupança) – Guarde dinheiro separadamente com opção de resgate.  
- Dark Mode – Interface com suporte a tema claro e escuro.

🔹 **Para Administradores**  
- Painel Administrativo – Visualização de todas as contas cadastradas.  
- Configuração de Contas – Definição de número da conta, agência e **Tipo de Conta** (Corrente, Poupança ou Mista).  
- Gestão de Limites – Definição de limites de crédito para contas Corrente e Mista.



🔗 **Tecnologias Utilizadas**

- **Back-end:** .NET 8 (ASP.NET Core Web API)  
- **Banco de Dados:** SQLite com Entity Framework Core  
- **Front-end:** HTML5, CSS3 (Variáveis, Flexbox, Grid) e JavaScript (Vanilla JS)  
- **Documentação da API:** Swagger
